# Inventory Management System

A **scalable, cloud-based Inventory and Supplier Management System** developed using modern **Full Stack Development technologies**. This project demonstrates expertise in **enterprise application development, RESTful API design, cloud deployment, and AI integration**.

Built with **Angular (Frontend)**, **Node.js with Express (Backend)**, and **MySQL (Database)**, and deployed using **ZOHO Catalyst Appsail** and **Clever Cloud**.

Additionally, the system integrates a **Retrieval-Based AI Agent** powered by **GROUNDX**, enabling intelligent insights from industry-standard inventory and supply chain resources.

##  Backend API Endpoints

### Inventory Operations

| Method | Endpoint             | Description                    |
|--------|--------------------|--------------------------------|
| GET    | `/inventory`        | Fetch all inventory items      |
| POST   | `/inventory`        | Create new inventory item      |
| GET    | `/inventory/:id`    | Fetch specific item            |
| PUT    | `/inventory/:id`    | Update item details            |
| DELETE | `/inventory/:id`    | Delete inventory item          |

### Supplier Operations

| Method | Endpoint             | Description                    |
|--------|--------------------|--------------------------------|
| GET    | `/suppliers`        | Fetch all suppliers            |
| POST   | `/suppliers`        | Create new supplier            |
| GET    | `/suppliers/:id`    | Fetch specific supplier        |
| PUT    | `/suppliers/:id`    | Update supplier details        |
| DELETE | `/suppliers/:id`    | Delete supplier                |

##  AI Retrieval System

The integrated AI agent retrieves insights from globally recognized inventory and supply chain management resources:

- *Supply Chain Management For Dummies* — John Wiley & Sons  
- *Inventory Management* — National Institute of Open Schooling  
- *Demand-Driven Inventory Optimization and Replenishment* — Robert Davis  
- *Essentials of Inventory Management* — Max Muller  
- *Logistics and Supply Chain Management* — Martin Christopher  

> The AI module enhances decision-making by providing intelligent, context-aware responses directly within the application.
